## 📁 Contenido de diseño

Este repositorio contiene la segunda versión del algoritmo del sistema **ClinData**, enfocada en el diseño e implementación de los algoritmos principales de captura clínica.

### Archivos principales

- **README.md**  
  Documento descriptivo del proyecto.

- **V2_CaptureAlgorithms.drawio.pdf**  
  Diagrama del flujo de captura de texto clínico.  
  Incluye:
  - Validación de abreviaturas
  - Corrección ortográfica
  - Sugerencias de autocompletado
  - Integración con CIE-10
  - Lógica de exportación del expediente (médico/paciente)

- **V2_ClinData_Algorithm.drawio.pdf**  
  Diagrama del flujo general del sistema ClinData.  
  Incluye:
  - Tipos de visita (consulta / urgencia)
  - Creación y búsqueda de expedientes
  - Manejo de primera consulta
  - Generación de notas clínicas
  - Flujo de triage y atención

### Propósito

Estos archivos representan el diseño lógico del sistema, sirviendo como base para la implementación del backend y frontend del expediente clínico inteligente.
